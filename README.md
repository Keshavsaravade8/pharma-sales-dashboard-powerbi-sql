# 💊 Pharma Sales Analytics Dashboard

## 📌 Project Overview
Analyzed pharma sales data using SQL and Power BI to generate business insights.

## 🛠 Tools Used
SQL | Power BI | Excel

## 📊 Key Insights
• 16–17% growth observed  
• N02BE is top-performing category  
• Seasonal demand patterns identified  
• Peak sales hours: 10 AM – 8 PM  

## 📂 Project Files
• SQL Queries  
• Power BI Dashboard  
• PPT Insights
